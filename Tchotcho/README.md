# تشوتشو مالح 🐦

لعبة Flappy Bird بطابع جزائري — طائر النورس الجزائري يطير فوق سماء العاصمة، بمقام النصب التذكاري (مقام الشهيد) في الخلفية.

## المميزات

- 🎮 طريقة لعب مطابقة لـ Flappy Bird الأصلي (اضغط للطيران، تفادى الأنابيب)
- 🎨 رسومات pixel art مخصصة (طائر النورس الجزائري بكل حالاته: طيران، سقوط، اصطدام)
- 🔊 أصوات وموسيقى خلفية مولّدة بالكامل بالكود (Web Audio API) — بدون أي ملفات صوتية خارجية
- 🏆 حفظ أفضل نتيجة وسجل الأرقام القياسية محلياً (localStorage)
- ⚙️ إعدادات لتفعيل/تعطيل الموسيقى والمؤثرات الصوتية
- 📱 متجاوب بالكامل (Responsive) - يعمل على الموبايل وسطح المكتب
- 🌐 دعم كامل للغة العربية (RTL)

## التقنيات المستخدمة

- React 19 + TypeScript
- Vite
- Tailwind CSS
- Web Audio API (للأصوات والموسيقى)
- localStorage (لحفظ النتائج)

## التشغيل محلياً

```bash
npm install
npm run dev
```

## البناء للإنتاج

```bash
npm run build
```

الملفات الناتجة تكون في مجلد `dist/`.

## النشر على Vercel

### الطريقة الأولى: عبر Vercel CLI

```bash
npm install -g vercel
vercel
```

اتبع التعليمات التي تظهر، واختر الإعدادات الافتراضية (Vercel يكتشف مشروع Vite تلقائياً).

### الطريقة الثانية: عبر لوحة تحكم Vercel (الأسهل)

1. ادخل إلى [vercel.com](https://vercel.com) وسجل دخول (يمكن عبر GitHub)
2. اضغط "Add New" → "Project"
3. ارفع هذا المجلد كـ repository على GitHub أولاً، أو استخدم خيار "Deploy" المباشر برفع الملف المضغوط
4. Vercel سيكتشف تلقائياً:
   - **Framework Preset:** Vite
   - **Build Command:** `npm run build`
   - **Output Directory:** `dist`
5. اضغط "Deploy" وانتظر بضع ثوانٍ

### الطريقة الثالثة: سحب وإفلات (الأسرع)

1. ادخل إلى [vercel.com/new](https://vercel.com/new)
2. اسحب مجلد المشروع كاملاً (بعد فك الضغط) إلى الصفحة
3. Vercel سيرفعه وينشره تلقائياً

## هيكل المشروع

```
flappy-game/
├── public/
│   ├── sprites/        # صور الطائر (pixel art)
│   ├── bg/              # خلفيات اللعبة وشاشة البداية
│   └── fonts/           # خطوط Press Start 2P و Tajawal (محلية)
├── src/
│   ├── components/      # مكونات React (Bird, Pipe, Ground, الشاشات...)
│   ├── game/             # محرك اللعبة (الفيزياء، الثوابت)
│   └── utils/            # أدوات الصوت وحفظ البيانات
└── vercel.json           # إعدادات النشر على Vercel
```

- **سطح المكتب:** اضغط بالماوس، أو استخدم مفتاح المسافة (Space) أو السهم العلوي (↑)
